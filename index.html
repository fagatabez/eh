Option 2 — full power
Contribute via worker.py
Run the worker script to contribute your full GPU. Detects your VRAM and auto-calculates the largest safe batch size.
  • Auto-calculates batch size from your VRAM
  • Live OOM probe — never crashes
  • Optional CPU sharing
Formula: vram_gb × 1024³ × max_vram% ÷ bytes_per_sample
Example (8 GB, 70%): 8 GB ÷ 32 MB/sample = ~178 batches
Setup — 3 steps
Step 1 — Install dependencies
pip install torch requests
Step 2 — Download worker.py
Step 3 — Run it
python worker.py
The worker detects your VRAM, calculates the max safe batch size, and does a live OOM probe. Press Ctrl+C to stop gracefully and return batches to the pool.
